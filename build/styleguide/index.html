Methodology

Code style rules

Class naming conventions

Um eine einheitliche Bennenung zu erreichen, wird BEM verwendet.. Dabei wird zwischen Blöcken, Elementen und Modifiern unterschieden.
Blöcke sind unabhängige Einheiten, die mit einem kurzen Klassennamen versehen werden:

<div class="site-search">...</div>

Elemente sind Teil eines Blocks und können nicht alleine verwendet werden:

<div class="site-search site-search__field">...</div>

Modifier verändern beispielsweise das Aussehen von Blöcken oder Elementen:

<div class="site search site-search__full">...</div>
